腎安守護

洗腎患者真正危險的時刻，不只是在透析室，而是在離開透析室後的返家與居家恢復期。

照護斷點

透析結束後常出現低血壓、心率異常、活動量下降與跌倒風險，但家庭與辦公室端缺少連續資訊。

核心設計

洗腎者佩戴裝置、床邊檢測器與居服員辦公室看板，將被照顧者狀態轉為可行動的即時健康指數。

預警流程

生命徵象、活動狀態與跌倒偵測轉換為健康指數，低於門檻時辦公室看板立即派員協助。

佩戴監測
床邊確認

Product System

三端式照護系統

  • 洗腎者佩戴裝置：顯示時間、心率、血壓與跌倒偵測。
  • 床邊檢測器：追蹤活動狀態，必要時報警求救並支援對話。
  • 居服員辦公室：多位被照顧者即時健康指數低於門檻時派員協助。

基礎風險程式

health = 100 - (bpRisk + heartRisk + activityRisk + fallRisk)
